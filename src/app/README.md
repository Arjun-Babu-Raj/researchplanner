# Research Planner

## About the Application

This application, **Research Planner**, is an interactive tool designed to assist students, researchers, and academics in drafting and refining a research study plan. Leveraging the power of generative AI, it helps structure your thoughts and generate key sections of a research proposal, from objectives to analysis plans.

## API Key Privacy

To use the AI features, you must provide your own Google Gemini API key. Please be assured that **your API key is not stored or saved in any way**. It is only held in your browser's memory for your current session and is used solely to make requests to the Gemini API on your behalf.
